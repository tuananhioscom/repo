Câu hỏi thường gặp

File in chuẩn là gì?
Sử dụng PDF/AI/PSD, ảnh 300dpi, chuyển CMYK, kèm bleed/crop mark. Nếu cần, gửi file nguồn – chúng tôi hỗ trợ xuất file đúng chuẩn.
Thời gian in & giao hàng?
Đơn tiêu chuẩn: 24–48 giờ. Đơn gấp vui lòng liên hệ trực tiếp qua Zalo/Điện thoại để ưu tiên.
Màu in có lệch so với màn hình?
Có thể có sai số do khác biệt hiển thị. Chúng tôi có bước duyệt proof/mẫu test để tối ưu đúng màu mong muốn.
